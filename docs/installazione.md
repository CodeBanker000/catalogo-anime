# Installazione

per installare il progetto è necessario avere installato git.

- Aprire il terminale
- Verificare che git sia installato:

```bash
git --version
```

- Clonare la repository:

```bash
git clone https://https://github.com/CodeBanker000/catalogo-anime

```

- Accedere alla cartella del progetto:

```bash
cd catalogo-anime
```

- Aprire il file `index.html` con un browser moderno (ad esempio Chrome, Firefox, Edge)
  - oppure, se si preferisce, è possibile utilizzare un server locale per servire i file (ad esempio con Live Server di Visual Studio Code)
